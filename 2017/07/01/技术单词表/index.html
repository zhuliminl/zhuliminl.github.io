技术单词表

2017-07-01

一直以来，我都对英文单词都不是特别敏感，很多英文能认识但是拼不出来。时间长了，我开始觉得有必要给自己做个专业的通用一点的词汇表，通过批量积累的方法，记住它们，借此来减少写代码的过程中一些不必要的脑力消耗。我会先从我熟悉的 UI 词汇开始，然后是 CSS 命名，最后是 JS 上面常用的以及一些更通用的计算机术语吧。不过我发现 UI 部分用不到的单词太多了，所以又把它排列到最后。这篇文章会不定时更新。
