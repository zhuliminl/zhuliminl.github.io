JavaScript 中的对象

2017-06-23

JavaScript中一共有六种主要数据类型。下面前5种为简单基本类型，而object则属于复杂数据类型。

JavaScript有许多特殊的对象子类型，我们可以称之为复杂基本类型。函数就是这个子类型之一，技术上可以说函数是可调用的对象。

  • string
  • number
  • boolean
  • null
  • undefined
  • object

内置对象

对象的子类型，通常被称之为内置对象。在JavaScript里面，它们实际上只是一些可以当作构造函数来用的内置函数。

  • String
  • Number
  • Boolean
  • Object
  • Function
  • Array
  • Date
  • RegExp
  • Error

属性描述符

从ES5开始，所有的属性都具备了属性描述符。下表是属性描述符的内容。

特性 描述 默认值
value 值
writable 可写 true
enumerable 可枚举 true
configurable 可配置 true

关于属性描述的方法。

方法 描述 参数
getOwnPropertyDescriptor() 拿到对象的属性描述符 对象，变量字符串
defineProperty() 自定义属性描述 对象，变量字符串， 描述字面量

不变性

有很多种方法来让对象不可改变。但要注意的是，这都是浅不变，也就是说只影响目标对象的直接属性，而不会影响目标对象中的其他引用内容。

效果 方法 特征
创建一个常量 writable :false, confiurable : true 不可修改、重定义、删除
禁止扩展 Object.preventExtensions(...) 不能再添加和修改
密封 Object.seal(...) 不能添加、修改、删除，还不能重新配置
冻结 Object.freeze(...) 最高级别的不变性

存在性

方法 能力范围 返回内容
hasOwnProperty() 只检查属性是否在对象中 布尔值
in 操作符 深入原型链中 布尔值
propertyIsEnumerable() 检查属性名是否在对象中并可枚举 布尔值
keys() 只检查对象直接包含并可枚举的属性 数组
getOwnPropertyNames() 只检查对象直接包含无论是否可枚举的属性 数组
